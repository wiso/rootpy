5.3. rootpy.io.TemporaryFile¶

class rootpy.io.TemporaryFile(*args, **kwargs)¶

A temporary ROOT file that is automatically deleted when closed. Uses Python’s tempfile.mkstemp() to obtain a temporary file in the most secure manner possible.

Positional and keyword arguments are passed directly to tempfile.mkstemp()
